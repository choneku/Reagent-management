라이브러리: html5-qrcode — Android Chrome · iOS Safari 지원
1
시약 등록 시 바코드 스캔
＋ 시약 등록 버튼 → 모달 상단 [📷 카메라] 버튼 탭
→ 핸드폰 전체화면으로 카메라 전환
→ 바코드를 박스 안에 맞추면 자동 인식
→ 인식 즉시 웹 화면으로 복귀
→ Lot번호 · 유효기간 · GTIN 자동 입력
2
스캔 대기열로 연속 스캔
헤더 [📷 스캔 대기열] 버튼 클릭
→ 담당자 이름 입력 → [조회] 클릭
→ [📷 카메라로 바코드 스캔] 버튼 탭
→ 전체화면 카메라 → 인식 후 자동 복귀 + Firebase 저장
→ 반복 탭으로 여러 시약 연속 스캔 가능
→ PC에서 대기열 조회 후 개별/일괄 등록

📷 카메라 스캔 사용법
핸드폰 또는 PC 카메라로 시약 바코드를 직접 스캔할 수 있습니다.
별도 앱 설치 없이 웹 브라우저에서 바로 사용 가능합니다.
1
카메라 버튼 탭
시약 등록 모달 → 바코드 스캔 영역의 📷 카메라 버튼 탭
스캔 대기열 모달 → 상단 📷 카메라로 바코드 스캔 버튼 탭
2
카메라 권한 허용
처음 사용 시 브라우저가 카메라 권한을 요청합니다.
허용을 탭하면 이후에는 자동으로 카메라가 열립니다.
※ HTTPS 환경(GitHub Pages)에서만 카메라 사용 가능
3
전체화면 카메라로 스캔
화면이 전체화면 카메라로 전환됩니다.
시약 바코드를 박스 안에 맞추면 자동 인식됩니다.
인식 성공 → 화면이 자동으로 웹으로 복귀합니다.
취소 시 우측 상단 ✕ 닫기 버튼을 탭하세요.
4
스캔 대기열 사용 시
카메라 스캔 전에 담당자 이름 입력 → [조회]를 먼저 눌러야
본인 대기열에 바코드가 저장됩니다.
여러 개 연속 스캔 가능 — 스캔할 때마다 대기열에 누적됩니다.
📌 인식 가능 바코드: GS1-128(선형) · DataMatrix(2D) · QR코드 · EAN-13
📌 인식 후 GS1 파싱: Lot번호 · 유효기간 · GTIN · 수량 자동 입력
📌 밝은 환경에서 스캔 시 인식률이 높아집니다
